아이라클

진정한 개인 맞춤 AI 사주 운세 플랫폼

현재 버전: v1.0.3
iOS: Build 41

앱 소개

아이라클(Airacle)은 전통 명리학의 정밀한 계산과 최신 GPT-4o AI 기술을 완벽하게 융합한 차세대 개인 맞춤 사주 운세 서비스입니다.

✨ 핵심 특징: 단순한 일반적인 운세가 아닌, 사용자 개인의 실제 사주 데이터를 완벽 분석하여 진정한 맞춤형 운세를 제공하는 업계 최고 수준의 개인화 플랫폼입니다.

주요 기능

🏠 신당 - 개인 운세 대시보드

일일/주간/월간/연간 운세, 행운요소, 점수 시스템, 개인화된 운세 알림

🧙‍♀️ AI CHAN - 개인 맞춤 상담

40년 경력 무당 수준의 개인 상담, 질문 카테고리 자동 분석, 대화 기록 저장

🎡 놀이터 - 궁합 & 행운번호

사주 기반 정밀 궁합 분석, 개인 사주 기반 로또 번호, AI 해석 정리

⚙️ 설정 - 계정 관리

개인정보 관리, 구독 관리, 토큰 충전, 알림 설정, 고객센터

기술 스택

React Native Expo SDK 53 Supabase GPT-4.1-mini PostgreSQL TypeScript Deno Runtime v4.0

개인정보 보호

아이라클은 사용자의 개인정보 보호를 최우선으로 합니다.

  • 모든 개인정보는 암호화되어 안전하게 저장됩니다
  • Row Level Security(RLS)로 데이터 접근을 제한합니다
  • JWT 토큰 기반 인증으로 보안을 강화합니다
  • 90일 후 자동으로 오래된 데이터를 정리합니다
  • 개인정보 처리방침을 통해 투명한 정보 수집을 보장합니다
