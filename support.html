❓ 자주 묻는 질문 (FAQ)

사주 분석은 어떻게 이루어지나요?

+

아이라클은 전통 사주학을 바탕으로 한 AI 분석 시스템을 사용합니다:

  • 생년월일시 입력: 정확한 출생 정보를 바탕으로 사주팔자 산출
  • 오행 분석: 목화토금수 오행의 균형과 특성 분석
  • 십신 관계: 개인의 성격과 운명적 특성 파악
  • 용신 분석: 개인에게 가장 필요한 기운 도출
  • AI 해석: 현대적 관점에서 이해하기 쉽게 해석 제공

궁합 분석은 어떤 기준으로 이루어지나요?

+

아이라클의 궁합 분석은 다음과 같은 전통 사주학 이론을 기반으로 합니다:

  • 오행 상생상극: 서로의 오행이 조화를 이루는지 분석
  • 간지 합충: 천간과 지지의 합(合)과 충(沖) 관계 확인
  • 십신 관계: 서로에게 미치는 십신의 영향력 분석
  • 용신 보완: 서로의 용신이 상호 보완되는지 검토
  • 종합 점수: 위 요소들을 종합하여 0~100점 궁합 점수 산출

운명 구슬은 어떻게 사용하나요?

+

운명 구슬은 아이라클의 주요 기능을 이용할 때 사용되는 포인트입니다:

  • 궁합 분석: 다른 사용자와의 궁합을 확인할 때 소모
  • 상세 분석 열람: 궁합 결과의 세부 항목을 볼 때 필요
  • 새로운 추천: 발견 탭에서 새로운 인연을 찾을 때 소모
  • 충전 방법: 운명구슬 충전소에서 구매 또는 프리미엄 구독 시 매월 150개 지급

일일 운세는 어떻게 확인하나요?

+

일일 운세는 프리미엄 기능으로 제공됩니다:

  • 접근 방법: 홈 화면 하단 탭에서 '오늘의 운세' 선택
  • 분석 내용: 개인 사주와 오늘의 일진을 바탕으로 한 맞춤 운세
  • 제공 정보: 운세 점수, 상세 조언, 행운의 숫자/색상, 키워드
  • 업데이트: 매일 새벽 자동 업데이트
  • 이용 조건: 프리미엄 구독 또는 7일 무료 체험 중 이용 가능

채팅은 어떻게 시작하나요?

+

채팅을 시작하는 방법은 다음과 같습니다:

  • 궁합 분석 후: 궁합 결과 화면에서 '채팅 요청' 버튼 클릭
  • 상대방 수락: 상대방이 채팅 요청을 수락해야 대화 가능
  • 채팅 제한: 무료 회원 3개, 프리미엄 회원 10개까지 동시 채팅방 유지
  • 친구 추가: 채팅 중 친구로 추가하여 지속적인 관계 유지 가능
  • 이미지 전송: 텍스트와 이미지 모두 전송 가능

프리미엄 구독의 혜택은 무엇인가요?

+

프리미엄 구독 시 다음과 같은 혜택을 받으실 수 있습니다:

  • 7일 무료 체험: 신규 가입 시 모든 프리미엄 기능 무료 체험
  • 오늘의 운세: 매일 개인 맞춤형 사주 기반 운세 제공
  • 운명 구슬 지급: 매월 150개 자동 지급
  • 채팅방 확장: 최대 10개까지 동시 채팅방 유지 가능
  • 거리 조절: 매칭 거리를 10km부터 전국까지 자유롭게 설정
  • 우선 고객지원: 문의사항에 대한 우선 처리

위치 정보는 어떻게 사용되나요?

+

위치 정보는 매칭 서비스 향상을 위해 사용됩니다:

  • 사용 목적: 근거리 사용자 추천 및 매칭
  • 정확도: 대략적인 지역 정보만 사용 (정확한 주소는 수집하지 않음)
  • 설정 변경: 앱 설정에서 위치 권한을 언제든 해제 가능
  • 거리 조절: 프리미엄 회원은 매칭 거리 범위 조절 가능
  • 개인정보 보호: 상대방에게는 대략적인 거리만 표시

계정을 삭제하고 싶어요

+

계정 삭제는 다음 방법으로 진행할 수 있습니다:

  • 앱에서 삭제: 마이페이지 > 계정 삭제 메뉴 이용
  • 고객센터 문의: airacleinfo@gmail.com으로 계정 삭제 요청
  • 삭제 처리: 요청 후 즉시 처리 (복구 불가능)
  • 데이터 보관: 관련 법령에 따라 일부 정보는 일정 기간 보관
  • 주의사항: 삭제 후에는 모든 데이터가 영구 삭제되어 복구할 수 없습니다
